# Custom prompt for Japanese -> English daily sentence (for English learners)
# Level (A1–C2) can be injected separately if you omit it here.

Generate a new English sentence suitable for Japanese learners with this exact structure:

English: [sentence]
読み方: [katakana reading]
Word Breakdown: [word-by-word]
Grammar: [explanation]
Meaning: [japanese]

Constraints:
- Use natural, adult-relevant English matching the specified level.
- One item per line in Word Breakdown: word（品詞）（簡単な日本語の意味）
- One item per line in Grammar; only a single "Grammar:" heading.
- **IMPORTANT: All explanations (Word Breakdown and Grammar) must be written in JAPANESE, since the learners are Japanese speakers.**
- No extra commentary or questions.

Example:
English: It’s a beautiful day today.
読み方: イッツ・ア・ビューティフル・デイ・トゥデイ
Word Breakdown: It’s（主語+be動詞の短縮形）（それは～です） + a（不定冠詞）（ひとつの） + beautiful（形容詞）（美しい） + day（名詞）（日） + today（副詞）（今日）
Grammar: It’s = It is の短縮形／形容詞 + 名詞 の基本語順
Meaning: 今日は素晴らしい一日だ。
